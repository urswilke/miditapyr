miditapyr.mido_io module¶

miditapyr.mido_io.compact_df(df, repair_reticulate_conversion=False)¶

Function to transform the dataframe returned by tidy_df() back to a format as in the result of midi_to_df()

Parameters

df – Dataframe returned by tidy_df()

Returns

a dataframe, containing 3 columns

Return type

pandas.DataFrame

The function miditapyr.mido_io.midi_to_df() returns a dataframe of the same format. This dataframe contains these columns:

  • i_track: the track number

  • msg: the (meta) event information read by mido.MidiFile() in a list of dictionaries

  • meta: whether the event in ‘msg’ is a mido meta event

miditapyr.mido_io.tidy_df(dfc)¶

Function to transform the dataframe returned by midi_to_df() in a tidy format (cf. https://r4ds.had.co.nz/tidy-data.html)

Parameters

dfc (pandas.DataFrame) – Dataframe returned by midi_to_df

Returns

a dataframe

Return type

pandas.DataFrame

The returned dataframe with the columns i_track and meta of midi_to_df. The msg column is exploded and each key in the dicts in the msg column of miditapyr.mido_io.midi_to_df() is stored in its own column.
